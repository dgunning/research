## Integration Checklist

### Before Presentations
- [ ] Identified audience's vocabulary level
- [ ] Selected appropriate domain terms
- [ ] Prepared simpler alternatives
- [ ] Defined essential jargon upfront
- [ ] Balanced technical precision with accessibility

### In Written Communication
- [ ] Used consistent terminology throughout
- [ ] Defined acronyms on first use
- [ ] Included glossary if needed
- [ ] Matched vocabulary to document purpose
- [ ] Avoided unnecessary jargon

### During Meetings
- [ ] Gauged room's technical level
- [ ] Adjusted vocabulary accordingly
- [ ] Clarified ambiguous terms
- [ ] Avoided alienating non-specialists
- [ ] Demonstrated expertise through precision
